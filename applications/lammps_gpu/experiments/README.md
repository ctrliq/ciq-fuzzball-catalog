

Scripts to set up example LAMMPS runs using tutorial data from
(https://github.com/lammpstutorials/lammpstutorials-inputs) in a standard format. Each experiment
directory contains a `stagein` script that creates the input files and a `lammps` script to run
LAMMPS on the inputs.

Available examples:

- Level1
  - [Lennard Jones fluid - first example](lj-simple)
  - [Lennard Jones fluid - improved input](lj-improved)
  - [Breaking a carbon nanotube - unbreakable bonds](cnt-unbreakable)
  - [Breaking a carbon nanotube - breakable bonds](cnt-breakable)
